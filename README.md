## ğŸš€ Getting Started

### Prerequisites
- Node.js 16 or higher
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/MasumRaza1/faangprep.git
cd faangprep
```

2. Install dependencies
```bash
npm install
# or
yarn install
```

3. Create environment file
```bash
cp .env.example .env
```

4. Start the development server
```bash
npm start
# or
yarn start
```
